OpenClaw

Instala OpenClaw en tu máquina local con un solo comando.
Sin tocar archivos. Sin conocimientos de Docker.

Sistema detectado automáticamente
// PowerShell (como Administrador)
PS C:\> irm https://raw.githubusercontent.com/nexxoads/openclaw-easy-install/main/install.ps1 | iex
Requisito: Tener Docker Desktop instalado y abierto.
WSL2 (recomendado): Si usas WSL2, abre tu terminal de Ubuntu/Debian y usa el comando de Linux.
// Terminal
$ curl -sSL https://raw.githubusercontent.com/nexxoads/openclaw-easy-install/main/install.sh | bash
Requisito: Tener Docker Desktop para Mac instalado y en marcha (icono de la ballena en la barra de menú).
// Terminal (Ubuntu / Debian / WSL2)
$ curl -sSL https://raw.githubusercontent.com/nexxoads/openclaw-easy-install/main/install.sh | bash
Requisito: Tener Docker instalado.
Si no tienes Docker: curl -fsSL https://get.docker.com | sh
// ¿Qué hace el instalador?
01

Detecta tu entorno

Comprueba automáticamente que Docker está instalado y en marcha. Si no lo está, te indica exactamente dónde descargarlo para tu sistema.

02

Crea la carpeta de datos

Genera ~/openclaw-local/ con todos los archivos de configuración necesarios. Tus datos siempre quedan en tu máquina.

03

Descarga la imagen oficial

Hace docker pull de la imagen ghcr.io/openclaw/openclaw:latest desde el GitHub Container Registry oficial de openclaw.ai.

04

Arranca el contenedor

Lanza OpenClaw con restart: unless-stopped — se vuelve a encender solo cada vez que reinicias el ordenador.

05

Muestra los siguientes pasos

Al terminar, aparecen todos los comandos que necesitas: configuración inicial, ver logs, parar, reiniciar y desinstalar.
